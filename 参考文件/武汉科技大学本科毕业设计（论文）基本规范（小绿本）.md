# 武汉科技大学本科毕业设计（论文）基本规范

**文件编号：** 武科大教发〔2024〕39 号
**发布机构：** 教务处编印（二〇二四年）

---

## 第一章 总则

**第一条** 毕业设计（论文）是培养学生综合运用所学知识，分析和解决实际问题，提高创新能力的重要环节。保质保量完成毕业设计（论文）是申请学士学位的基本要求。为进一步提高我校本科毕业设计（论文）质量、学士学位授予质量，根据《武汉科技大学普通本科毕业设计（论文）工作管理办法》，就我校本科毕业设计（论文）的撰写格式、印刷与装订等统一制定本基本规范。

---

## 第二章 结构及要求

### 第二条 组成部分

一般各学科专业的毕业设计（论文）由以下部分组成：

- 毕业设计（论文）封面
- 中文摘要
- 英文摘要
- 目录
- 正文
- 参考文献
- 致谢
- 附录
- 有关图纸（大于 3 号图幅时单独装订）

**艺术类专业**（毕业设计/创作）：与上述要求类同。以下内容凡未特别说明，则视毕业设计（创作）为理工类毕业设计（论文），相应的要求均适用于毕业设计（创作）。

**医科类专业**（毕业论文）：由封面、中文摘要、英文摘要、目录、正文、参考文献、致谢、附录、综述、综述参考文献组成。

**终稿格式**：统一提交 PDF 格式。

### 第三条 封面

封面由学校统一格式（可在本科生院网下载）。

- 题目一般不得超过 **20** 个字（医科类专业不超过 **25** 个字），要简明扼要，可加副标题。

### 第四条 摘要

- 摘要应尽量反映毕业设计（论文）的主要信息，内容包括**研究目的、方法、结果和结论**。
- 不含图表，不加注释，具有独立性和完整性。
- 中文摘要一般为 **400 字左右**，英文摘要应与中文摘要内容一致。
- **关键词**：一般为 **3-5 个**，是反映毕业设计（论文）主题内容的名词。

### 第五条 目录

- 按**三级标题**编写，层次清晰，且要与正文标题一致。
- 主要包括正文、参考文献、致谢等。
- **中英文摘要在目录中不出现**。

### 第六条 正文

正文部分包括：**绪论（前言、序言）、主体及结论（结束语）**。

- **绪论**：简要说明选题的目的和意义，国内外文献综述（医科类另列附正文后），以及所要研究的内容。
- **主体**：主要组成部分，要求层次清晰，文字简练，通顺，重点突出。
- **正文格式**：小四号宋体字（西文及阿拉伯数字用 Times New Roman 字体），行间距 **1.25 倍**。

#### 题序层次格式

##### 表 1：理工科类设计（论文）、医科类和外语类论文层次代号及说明

| 层次 | 格式 | 说明 |
|------|------|------|
| **章**（第一层次） | `1□×××××` | 小二号黑体字，**顶格**，前面空 1 行（小四号字）。题序和标题之间空两个字符，不加标点，**左对齐**。 |
| **节**（第二层次） | `1.1□×××××` | 小三号黑体字，**顶格**。题序和标题之间空两个字符，不加标点，**左对齐**。 |
| **条**（第三层次） | `1.1.1□×××××` | 四号黑体字，**顶格**。题序和标题之间空两个字符，不加标点，**左对齐**。 |
| **项**（第四层次） | `1.1.1.1□×××××` | 小四号黑体字，**顶格**。题序和标题之间空两个字符，不加标点，**左对齐**。 |
| 五级 | `1.1.1.1.1□×××××` | 同上 |

**注：** 西文及阿拉伯数字，不用中文黑体字，而用 **Times New Roman** 字体。

##### 表 2：文科类（不含外语类）论文层次代号及说明

| 层次 | 格式 | 说明 |
|------|------|------|
| **章**（第一层次） | `□□一、×××××` | 小二号黑体字，前面空 1 行（小四号字）。题序前空两格；题序和标题之间为顿号，标题后不加标点，左对齐，行首标点压缩。 |
| **节**（第二层次） | `□□（一）×××××` | 小三号黑体字。题序前空两格；题序用中文半角状态下的小括号，与标题之间不空格，标题后不加标点，左对齐。 |
| **条**（第三层次） | `□□1．×××××` | 四号黑体字。题序前空两格；题序后为西文全角状态下的句号"．"，与标题之间不空格，标题后不加标点，左对齐。 |
| **项**（第四层次） | `□□（1）×××××` | 小四号黑体字。题序前空两格；题序和标题之间不空格，括号、圈号等均为中文半角状态，标题后不加标点，左对齐。 |
| 五级 | `□□①×××××` | 同上 |

**注：** 西文及阿拉伯数字，不用中文黑体字，而用 **Times New Roman** 字体。

#### 结论

- 论文写结论，设计写结束语。
- 阐述作者创造性的工作，对存在的问题和不足应给予客观的说明，也可提出进一步的设想。
- 整个论文（设计）的总结，应以简练的文字说明论文（设计）的内容；用准确和精练的语言表述所得到的结论。

### 第七条 参考文献

- 列出自己在阅读并在文中引用的公开发表在出版物上的主要文献或网上下载的资料。
- 引用序号置于所引用部分的右上角（如 `[1]`），以尾注形式在文后按顺序一一列出。
- 文科类毕业论文也可在文中以**脚注**形式标明，每页脚注序号均从①开始。
- **数量要求**：
  - 理工类：不少于 **15** 篇（外文不少于 **3** 篇）
  - 文科类：不少于 **20** 篇（外文不少于 **3** 篇）
  - 医科类正文：不少于 **15** 篇（外文不少于 **3** 篇）；综述：不少于 **15** 篇（外文不少于 **6** 篇）
  - 毕业设计：不少于 **5** 篇（外文不作硬性要求）
  - 艺术类：不作硬性要求
- 同一参考文献引用多次只视为一篇，序号应一律为第一次出现时的序号。

### 第八条 致谢

对导师和给予指导或协助完成毕业设计（论文）工作的组织和个人表示感谢，内容应简洁明了、实事求是，避免俗套。

### 第九条 附录

对于不宜放在正文中的重要支撑材料，可编入附录中。包括：重要的原始数据、详细数学推导、程序全文及其说明、复杂的图表等一系列需要补充提供的说明材料。

### 第十条 其他相关要求说明

#### （一）文字

严格执行文字规范。正文字数要求：

| 类别 | 最低字数 |
|------|----------|
| 理工科类 | ≥ 10000 字 |
| 文科类 | ≥ 12000 字（外语专业 ≥ 4000 单词，体育类 ≥ 8000 字） |
| 艺术类 | 设计说明书 ≥ 5000 字，美术创作类论文 4000-5000 字 |
| 医科类 | 正文 ≥ 6000 字，综述 ≤ 3000 字 |

#### （二）插表

- 表格一般采取**三线制**：不加左、右侧边线，上、下底边线为粗实线（**1.5 磅**），中间为细实线（**0.75 磅**）。比较复杂时可适当增加横线和竖线。
- 表序**按章编排**（如 `表 1.1`、`表 2.5`）。表序与表名之间空一格，表名不使用标点符号。**表序与表名居中置于表上**，采用**五号宋体字**。
- 表头设计应简单明了，尽量不用斜线。表头中可采用化学符号或物理量符号。全表如用同一单位，将单位符号移到表头右上角，加圆括号。
- 表中数据应正确无误。数字空缺的格内加"—"字符（占 2 个数字宽度）。
- 表内相同文字和数字不允许用 `″`、`同上` 之类的写法，可采用通栏处理方式。
- 文科类论文插表下可增列补充材料、注解等。补充材料中文字用宋体小五号字，外文及阿拉伯数字用 Times New Roman 体小五号字。
- 表序、表名、插表及插表下增补材料等内容为一整体，**不得拆开排于两页**。插表应编排在正文提及处之后；该页空白处不够时，可将其后文字部分提前排写，将插表移到次页最前面。

#### （三）插图

- 应符合国家标准或行业标准，与文字紧密配合，文图相符，技术内容正确。
- 图序**按章编排**（如 `图 1.1`、`图 2.5`）。仅有一图时，在图题前加"附图"字样。
- 图序号、图中文字用**五号宋体**。
- 由若干个分图组成的插图，分图用 **a, b, c, ……** 标出。
- **图序和图题置于图下方中间**。插图与图序、图题为一整体，**不得拆开排于两页**。插图应编排在正文提及处之后；该页空白处不够时，可将其后文字部分提前排写，将图移到次页最前面。

#### （四）公式

- 公式原则上**居中**书写。若公式前有文字（如"解"、"假定"等），文字顶格书写，公式仍居中写，公式末**不加标点**。
- 公式序号**按章编序**（如 `(1.1)`、`(2.5)`）。公式序号排在版面右侧，且距右边距离相等。
- 文中引用公式时，一般用"见式（1.1）"或"由公式（1.1）"。
- 公式较长时在等号"＝"或运算符号"＋、－、×、÷"处转行，转行时运算符号书写于转行式前，不重复书写。
- 分式注意事项：繁分式分数线的长短（主、副分数线严格区分），主分数线与等号对齐。

**字体规范：**

| 类型 | 字体 | 示例 |
|------|------|------|
| 计量单位（如 kg）、固定标准函数（如 sin(·)）、事物名称、数字、数学运算符 | **正体** | `kg`, `sin`, `log` |
| 外文变量或物理量、非固定标准函数 | **斜体** | `F = Ma sinθ`, `F, M, a, θ` |
| 向量、集合、矩阵、张量 | **粗斜体** | `α`, `β` 向量 |

一般应为西文半角状态下的 **Times New Roman** 字体。

#### （五）数字用法

- 公历世纪、年代、年、月、日、时间和各种计数、计量，均用**阿拉伯数字**（特殊情况如直接引用的原文为汉字表述除外）。
- 年份不能简写（如 1999 年不能写成 99 年）。
- 数值的有效数字应全部写出（如 `0.50:2.00` 不能写作 `0.5:2`）。
- 全文有效数字应统一。

#### （六）软件

软件流程图和原程序清单要按软件文档格式附在毕业设计（论文）后面，特殊情况可在答辩时展示，不附在毕业设计（论文）内。

#### （七）图纸

- 图纸的设计、装订等应符合国家标准或行业标准。
- 图幅 ≤ 3 号图幅时应装订在论文内；> 3 号图幅时按国标规定单独装订作为附图。

#### （八）计量单位

计量单位的定义和使用方法按国家计量主管部门规定执行。

#### （九）附录的层次、图、表、公式

- 附录依次用"附录 A、附录 B、附录 C……"表示。
- 附录内的分级序号采用"附 A1、附 A1.1、附 A1.1.1"等表示。
- 图、表、公式依此类推为"图 A1、表 A1、(A1)；图 B1、表 B1、(B1)"等。
- 附录的图、表、公式的其他要求同正文。

### 第十一条 外文翻译

- 除外语类、艺术类专业外，其他各专业要求每人根据原文**连续翻译 1.0 万字符以上**外文文献。
- 不允许跳译，所翻译的外文文献必须与本论文议题相关。

## 第三章 印刷与装订

### 第十二条 印刷规格

| 项目 | 要求 |
|------|------|
| 纸张 | **A4** 打印 |
| 页边距 | 上下左右均为 **2.5cm** |
| 页眉距边界 | **2cm** |
| 页脚距边界 | **1.75cm** |
| 页眉文字 | "武汉科技大学本科毕业设计" 或 "武汉科技大学本科毕业论文" |
| 页眉字体 | **三号隶书**，居中，单倍行距 |
| 段前间距 | 每页首行文字段前间距为 **0.5 行** |
| 装订 | 一律**左侧装订** |
| 页码（摘要/目录） | 罗马字母，小五号字，**底端居中** |
| 页码（正文） | 阿拉伯数字，小五号 Times New Roman 字体，**底端居中** |

**双面打印注意事项：** 正文须另页右面开始，全文以右页为单页页码，不得有页码错误。右面、右页指文本正放时页面朝上的书脊的右面、右页。

### 第十三条 装订顺序

毕业设计（论文）按以下顺序编排合订成册：

1. 毕业设计（论文）封面（格式见附件 1，可在本科生院网下载）
2. 中文摘要、关键词（格式见附件 2）
3. 英文摘要、关键词（格式见附件 3）
4. 目录（格式见附件 4）
5. **正文**（题目在正文前不出现）
6. 参考文献（格式见附件 5）
7. 致谢（另起一页，标题用三号黑体居中；标题与内容之间空一行；内容用小四号宋体，1.25 倍行距，页面设置同正文）
8. 附录
9. 有关图纸（医科类为综述、综述的参考文献）
10. 封底

### 第十四条 外文翻译版式与装订

- A4 纸型，单独装订成册。
- 具体要求见附件 6《外文翻译格式要求》、附件 7《外文翻译封面格式》、附件 8《外文翻译首页格式（参考）》。

### 第十五条 资料存档顺序

**理工/文科/医科类：**

1. 毕业设计（论文）手册
2. 毕业设计（论文）
3. 外文翻译
4. 答辩记录单
5. 成绩单
6. 其它（如单独装订的图纸等）
7. 查重报告

**艺术类：**

1. 毕业设计（创作）手册
2. 毕业设计（创作）
3. 答辩记录单
4. 成绩单
5. 图册（单独装订的图纸等）
6. 展板小样（A3 幅面）
7. 查重报告

## 第四章 基本规范的执行

**第十六条** 各学院（部）毕业设计（论文）领导小组，负责组织对本学院关于本规范的执行情况进行检查。

**第十七条** 指导教师必须按本规范审查学生的毕业设计（论文），并在答辩前完成，审查合格后方可参加答辩；审查不合格者，应及时修改，达到合格后，才能参加答辩。

## 附件 1：毕业设计（论文）封面格式

### 本科毕业设计封面

- **题目**：居中，用**一号楷体字**，西文及阿拉伯数字用 Times New Roman 字体
- 格式全校统一，封面可在本科生院网下载，用本科生院发的 **A3 幅面纸**打印
- 以下六项均为**四号字体**（学号为 Times New Roman 字体，其余为宋体）：
  - 学院:
  - 专业:
  - 学号:（Times New Roman 字体）
  - 学生姓名:
  - 指导教师:
  - 日期:（示例：二○二三年六月）

### 本科毕业论文封面

- 同上，封面标题为"本科毕业论文"

## 附件 2：中文摘要格式要求

- 前面与页眉之间空一行
- **标题"摘要"**：小二号黑体，居中。空两格居中
- 空一行
- **内容**：小四号宋体，**1.5 倍行距**。摘要要求四段式：**目的、方法、结果、结论**。
- 空一行
- **关键词**：四号黑体，左对齐，后加冒号。内容用小四号宋体，词之间用分号分隔。
- 页码：用罗马字母小五号字底端居中。

---

## 附件 3：英文摘要格式要求

- 与页眉之间空一行
- **标题"Abstract"**：小二号 Times New Roman，居中，加粗
- 空一行
- **内容**：小四号 Times New Roman，1.5 倍行距
- 空一行
- **Key words**：四号 Times New Roman 字体，左对齐，加粗，后加冒号。内容用小四号 Times New Roman，首字母大写，关键词之间空两个字符。
- 页码：用罗马字母小五号字底端居中，接前页连续编序。

```
附录 B  ※※※※※※※※................................................28
```

- 页码：用罗马字母小五号字底端居中，接前页英文摘要的页码连续编序。

### 文科类（不含外语类）

- 同上，题序格式按文科类要求书写。

## 附件 5：参考文献格式要求

- 参考文献的排列应按照**引用的先后顺序**进行。
- 作者人数在 3 人以下（含 3 人）的全部列出，超过 3 人时一般只列出 3 名作者，后面加"等"字。不同作者姓名间用逗号隔开。
- 姓名一律采用"**姓在前名在后**"的写法，外文姓名按国际惯例缩写，并省略缩写点，空一个字符。
- 未公开发表的资料一般不能引用，确有必要须以脚注形式标注。
- 中文参考文献著录格式中的句号用西文全角状态下的"．"，西文参考文献著录格式中的标点符号用西文状态下的符号，后空一格。
- **标题"参考文献"**：另起一页，三号黑体居中；标题与内容之间空一行；内容用小四号宋体，1.25 倍行距，页面设置同正文。

### 常用参考文献编写规则

#### 1. 著作图书类
```
[序号]□作者．书名[M]．（版次，第 1 版可以不写版次）．出版地：出版者，出版年．引用起～止页
```
例：
```
[1] 龙驭球，包世华主编．结构力学[M]．（第 2 版）．北京：高等教育出版社，1994.17~20
[2] Zienkiewicz O. C. The Finite Element Method[M]. (3rd Ed). New York: McGraw–Hill, 1997. 21~27
```

#### 2. 翻译图书类
```
[序号]□作者．书名[M]．译者．版次（第 1 版可以不写版次）．出版地：出版者，出版年．引用起～止页
```
例：
```
[3] 保罗·萨缪尔森，威廉·诺德豪斯．经济学[M]．萧琛译．第 17 版．北京：人民邮电出版社，2004.105~115
```

#### 3. 学术刊物类
```
[序号]□作者．文章名[J]．学术刊物名，年，卷（期）：引用起～止页
```
例：
```
[4] 陈勇．中国保健食品的发展方向探索[J]．中国卫生事业管理，2003，19（1）：54~55
[5] Marshall A. An investigation into the mechanism of inhibition of a synergistic dianodic corrosion inhibitor[J]. Corrosion-Nace, 1981, 37(4): 214
```

#### 4. 学术会议类
```
[序号]□作者．文章名[C]．编者名．会议名称，会议地址，年份．刊物名．出版地：出版者，出版年：引用起～止页
```
例：
```
[6] 张浩．CAD/CAM 技术现状及发展综述[C]．中国通信学会等．第六届全国计算机应用联合学术会议，北京，2002．第六届全国计算机应用联合学术会议论文集．北京：北京邮电大学出版社，2002：3~108
```

#### 5. 学位论文类
```
[序号]□学生姓名．学位论文题目[D]．学校及学位论文级别．答辩年份．引用起～止页
```
例：
```
[7] 莫祖栋．广义集成优化反演法在机械设计中的应用研究[D]．湖南大学硕士学位论文．2006．14~27
```

#### 6. 报纸文献
```
[序号]□作者．文章名[N]．报纸名，出版日期（版别）
```
例：
```
[8] 李斌，吴晶晶．我国已建立一百三十个流浪儿童救助保护中心[N]．人民日报，2005-8-18（10）
```

#### 7. 在线文献
```
[序号]□作者．文章名．电子文献的出处或可获得地址，发表或更新/引用日期
```
例：
```
[9] 彭国华，叶帆．科学发展观是我们党对发展规律认识的新的飞跃．http://theory.people.com.cn/GB/40553/4239263.html，2007-1-10
```

## 附件 6：外文翻译格式要求

### 一、封面格式

外文翻译封面格式见附件 7（可在本科生院网下载）。

### 二、内容要求

- 版面页边距上下左右均为 **2.5cm**；页眉距边界 **2cm**；页脚距边界 **1.75cm**
- 页眉加"武汉科技大学本科毕业设计外文翻译"或"武汉科技大学本科毕业论文外文翻译"，字体为**三号隶书**，居中，单倍行距
- 每页首行文字段前间距为 **0.5 行**
- 一律**左侧装订**
- 页码用阿拉伯数字，小五号 Times New Roman 字体，底端居中

#### （一）首页

- **第一项**：外文论文题目，作者．学术刊物名，年，卷（期）：引用起～止页（如果是著作为：外文著作书名，翻译部分的章节序号，章节名，作者．版次．出版地：出版者，出版年．引用起～止页）——**原文书写**。
- **第二项**：同上内容——**中文翻译**。
- 首页格式见附件 8。

#### （二）翻译后的中文正文

- 正文按原文的章节结构顺序进行翻译。正文用**小四号宋体字**（西文及阿拉伯数字用 Times New Roman 字体），行间距 **1.25 倍**。
- 一级标题（标题号 1）：**小三号黑体字**，单倍行距，左对齐，段前段后空 0.5 行
- 二级标题（标题号 1.1）：**四号黑体字**，单倍行距，左对齐，段前段后空 0.5 行
- 三级标题（标题号 1.1.1）：**小四号黑体字**，单倍行距，左对齐，段前段后空 0.5 行
- 四级标题（标题号 1.1.1.1）：**小四号宋体字加粗**，单倍行距，左对齐，段前段后空 0.5 行

#### （三）外文原文附件

外文原文作为附件，放在翻译后的中文正文之后，另起一页。应为刊物电子版原版的打印件或刊物复印件，A4 纸型，要清晰、整洁。

### 三、打印及装订

- 封面用普通 **A3 纸型白纸打印**，对折装订；内容用 A4 纸打印。
- 一律**左侧装订**。
- 装订顺序：①封面 ②首页 ③翻译后的中文正文 ④待翻译前的外文原文 ⑤封底

## 附件 7：外文翻译封面格式

- 封面标题"本科毕业设计外文翻译"或"本科毕业论文外文翻译"
- 外文译文题目（中文）：题目居中，用**三号楷体字**，西文及阿拉伯数字用 Times New Roman 字体
- 格式全校统一，可在本科生院网下载
- 以下六项均为**四号字体**（学号为 Times New Roman 字体，其余为宋体）：
  - 学院:
  - 专业:
  - 学号:（Times New Roman 字体）
  - 学生姓名:
  - 指导教师:
  - 日期:（示例：二○二三年六月）
