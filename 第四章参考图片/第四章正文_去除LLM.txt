

         第4章 基于课程学习的训练优化方法


   前文通过强化学习实现了四足机器人基本运动目标的实现，但四足机器人
需要应对的环境复杂，且对于运动性能也有一定要求，因此需要在训练过程中
提高四足机器人的综合运动能力。通过大范围随机化运动控制目标或者随机化
训练地形的方法，会让智能体在早期策略不成熟的情况下难以从复杂样本中稳
定学习，因此需要应用课程学习的方法，让机器人逐渐适应由易到难的学习过
程。本章在训练过程中设置了速度指令课程以及地形课程，分别增强四足机器
人应对高速目标以及复杂地形的能力，并进一步融合两种课程，提高强化学习
过程的采样效率与效果。


4.1 课程学习理论基础

   对于一个机器学习过程，策略的学习过程如果有据可依，比如将复杂任务
按规律由易到难地分解，那么神经网络可以更快地学习到特征，也会更容易学
习到困难的任务目标。在训练神经网络的过程中，同样可以依照一定的难易顺
序，先采样简单的数据进行拟合，训练效果达到期望水平后再去采样复杂的数
据进行学习。课程学习[79]（Curriculum Learning, CL）通过权重区分不同难度样
本大采样频率，最先给简单样本以高权重，让简单样本拥有更大的概率被训练，
接着将训练起来较为困难的样本按照学习进程逐渐提高权重，最终让网络学习
到了高难度的特征。
   在课程学习中，对于任意表示训练样本集的随机变量 z ，其目标学习到的
概率分布映射函数为 P ( z ) ，其进行到第 λ 步（ λ ∈ [ 0,1] ,表示该优化问题的难易
程度， λ 越大则代表优化越困难，是一个单调递增序列）时的权重为 Wλ ( z ) ，
则第 λ 步时 z 的训练样本分布 Qλ ( z ) 为：

                     Qλ (z) = Wλ (z)P(z) ∀z     (4-1)

   同时，对于最终过程即 λ = 1 时，有∫ 𝑄 (𝑧)𝑑𝑧 = 1，即：

                       Q1(z) = P(z) ∀z          (4-2)



  式(4-2)表示，当𝜆 = 1即整个训练过程结束时，训练分布 𝑄 (𝑧) 等于目标分
布 𝑃(𝑧)，如果这些分布的熵随着 λ 的增加而增加，则这个分布序列 𝑄 被称为课
程，即：

                H(Qλ ) < H(Qλ+ò) ∀ò > 0    (4-3)

式中 𝐻 表示熵，熵的增加意味着分布变得更加均匀或“不确定”，同时在𝜆增加
 过程中，有：

              Wλ+ò(z) ≥Wλ (z) ∀z, ∀ò > 0   (4-4)

  上式表明随着 λ 的增加，样本权重𝑊 (𝑧)是单调递增的。该式描述了如何系
统地组织学习过程，使得学习者可以从简单的概念开始，逐渐过渡到更复杂的
概念，从而提高学习的效率和效果。通过增加样本权重𝑊 (𝑧)和熵𝐻，可以让学
习者在学习过程中被引导去关注更多样的样本和概念。
  课程学习的关键是通过难度测量器作为分级函数对每个任务给出其学习优
先程度，同时通过训练调度器决定该难度下的采样规则。“难度测量器+训练调
度器”的结构式课程学习中最常见的基本模式。课程学习可以分成两中类别：
预定义课程与自适应课程：

4.2 单一课程策略设置方法

4.2.1 优化速度性能的课程策略及其验证

  通过基于动力学模型的控制算法实现四足机器人高速运动往往需要精细考
虑动力学模型、高动态运动特性以及控制算法鲁棒性，而通过强化学习实现高
速运动则需要考虑如何让智能体适应巨大的速度变化范围并稳定学习到优秀策
略。在训练过程中，四足机器人作为智能体实机上是通过追踪不同的速度指令
来学习一个以速度为条件的策略。在强化学习的每个回合𝑘中，纵向和偏航速
度指令根据概率分布进行采样。由于四足机器人通过纵向速度与横摆角速度就
可以灵活实现 z 平面上的全向运动，因此横向速度指令则单独从一个较小的均
匀概率分布中采样。在没有课程学习的情况下，从一个回合到下一个回合的采
样过程保持不变，该过程可以表示为如下公式：

                   pvk x+,1ω z (⋅, ⋅) ← pvkx ,ω z (⋅, ⋅)           (4-5)

  在训练初期，如果速度指令从一个小范围内均匀采样，智能体可以很快学
会速度跟踪的任务目标。但如果指令速度从一个较大的分布中均匀采样，训练
会因为策略更新混乱而失败。失败的原因是高速下的运动控制具有挑战性，如
果前期无法完成的高速指令过多，智能体将无法获得足够的奖励。这个问题可
以通过先让智能体接触低速指令，然后通过课程逐渐增加所需的速度来缓解。
因此可以采用课程学习的方式，让速度指令根据时间变量𝑘在固定的时刻更新，
该更新规则𝑓可以表示为：

               pvkx+,1ωz (⋅, ⋅) ← f ( pvkx ,ωz (⋅, ⋅), k )         (4-6)

  采用预定义课程安排需要手动调整，且如果环境或学习算法发生变化，智
能体的学习速度将发生变化，将需要重新调整课程计划。为了让课程在强化学
习学习过程中更加灵活，可以通过奖励值的大小来自动决定课程的策略更新[57]。
考虑纵向线速度v 和横摆角速度ω 的独立分布𝑝 (⋅)和𝑝 (⋅)，采样过程为：

                  p v x , ω z ( ⋅, ⋅ ) = p v x ( ⋅ ) p ω z ( ⋅ )   (4-7)

  并为线速度与角速度分别指定单独的更新规则𝑓 , 𝑓 ：



                              pvkx+1 (⋅) ← f v ( pvkx (⋅), rvx )                                     (4-8)

4.3 速度-地形双课程融合方法

4.3.1 双课程融合策略设计

  前文采用课程学习的方法分别增加了策略模型对大范围速度分布与地形分
布的学习能力，如果同时让智能体学习速度指令课程以及地形课程，就能让四
足机器人机动性能全方位提高。速度指令课程中的纵向线速度指令课程与横摆
角速度指令课程通过联合概率密度分布的方法同时探寻两种运动目标的综合最

4.4 本章小结

  本章首先设计了关于控制指令速度的课程，通过纵向线速度与横摆角速度
的联合概率决定四足机器人运动控制指令更新方向；同时设计了关于地形适应
能力的课程，根据四足机器人该回合的区域内移动距离决定地形难度的升降。
对于地形适应能力，针对本体感受四足机器人额外设定了地形估计的特权状态
空间设置。对于两种课程，为了保证采样效率与学习效果，通过指令采样范围
融合了两种课程学习方法，在训练过程中进一步
优化采样效率。
